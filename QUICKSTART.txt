RUNNING LOCALLY (Windows)
============================================================

   python generate_mock_data.py

   python main.py

3. Check Output:

   - Chart Image: Output/chart.png

============================================================
OPTIONAL: Add Gemini API Key for Real AI Insights
============================================================

1. Get API key from: https://makersuite.google.com/app/apikey

2. Edit .env file (currently ignored for security):
   GEMINI_API_KEY=your_actual_api_key_here

3. Re-run: python main.py

Note: The pipeline works WITHOUT the API key using smart fallback analysis.

============================================================
DOCKER DEPLOYMENT (Production)
============================================================

1. Ensure Docker is installed

2. Build and run:
   docker-compose up --build

3. Output persists to local Output/ folder via volume mounts

============================================================
WHAT WAS FIXED
============================================================

Problem 1: pandas==2.1.4 and scikit-learn==1.3.2 
- These versions don't have pre-built wheels for Python 3.13
- Required C++ compiler (MSVC) which you don't have

Solution: Updated to newer versions:
- pandas >= 2.2.0
- scikit-learn >= 1.4.0
- These have pre-compiled wheels for Python 3.13

Problem 2: WeasyPrint requires GTK system libraries on Windows
- Very difficult to install on Windows

Solution: Switched to ReportLab
- Pure Python PDF library
- No system dependencies
- Works perfectly on Windows
